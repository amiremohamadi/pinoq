# pinoq
pinoq (/pɪˈnoʊki.oʊ/, like pinocchio) is a [deniable encrypted](http://en.wikipedia.org/wiki/Deniable_encryption) filesystem, built on top of FUSE.

## requirements
on ubuntu:
```sh
$ sudo apt-get install -y libfuse-dev libssl-dev pkg-config
```
